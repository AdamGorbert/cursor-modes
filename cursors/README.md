# Cursor images

Drop your cursor images here, then point each mode at them in
**Game Settings → Configure Settings → Cursor Modes → Configure Cursors**.

- Use a non-animated image (`.png`, `.avif`, `.webp`, or `.cur`).
- Keep it small: max 128×128, **32×32 recommended**.
- Set the hotspot (click point) in the settings menu — `16,16` centres a 32px image.

These files are referenced by path (e.g. `modules/cursor-modes/cursors/target.png`),
so anything in your Foundry `Data/` directory works too.
